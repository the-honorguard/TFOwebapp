# APP-6E squad markers

These SVG markers were generated with
[Python Military Symbols](https://github.com/nwroyer/Python-Military-Symbols)
version 2.0.8, using its `light` style and transparent canvas.

The generator is MIT-licensed. The assets use friendly land-unit symbols from
the NATO APP-6E / MIL-STD-2525 family. Filenames use an `app6-` prefix so they
can coexist with the earlier specialized PNG marker set in `images/icons`.
