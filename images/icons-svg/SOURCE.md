# APP-6E specialized markers

These SVG markers are standardized counterparts to the original PNG markers
in `images/icons`. They were generated with
[Python Military Symbols](https://github.com/nwroyer/Python-Military-Symbols)
version 2.0.8, using its `light` style and transparent canvas.

The generator is MIT-licensed. The generated symbols use friendly land-unit
symbols from the NATO APP-6E / MIL-STD-2525 family.
